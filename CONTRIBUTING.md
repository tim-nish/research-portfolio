# Contributing

Issue flow: open issues are classified with `triage:*` labels by the owner's
/triage-gh pipeline — `triage:direct` issues are implemented straight from the
issue body (one issue → one branch → one PR), `triage:story` issues get a bound
story file under `docs/stories/`, and `triage:spec` issues amend the governing
spec in `docs/` before decomposition.
