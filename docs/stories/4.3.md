## Gate

## Story

As a Japanese developer-community reader (P4), I want a lightweight JP landing
surface, so that I have a native-language bridge into the owner's work without a full
translated site existing.

## Acceptance criteria (once un-gated)

1. `/ja/` is a single page (not a route tree), presenting identity + a bridge to Zenn
   and to the (English) hub — consistent with AP-11 ("first-class content attribute,"
   not a parallel site).
2. No machine-translated mirrors are introduced anywhere on the site as part of this
   work (permanent non-goal, spec §11) — this story must not scope-creep into
   translating existing English pages.
3. URL space `/ja/` was already reserved since Phase 1 (spec §7.1) and unsquatted.
